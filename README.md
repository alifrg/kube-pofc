# kube-pofc
Kuberentes Cluster proof of concept project

# QuickStart
```
$ git clone https://github.com/alifrg/kube-pofc.git
$ cd kube-pofc
$ vagrant up
```

# Dependencies
- ansible (Tested against version 2.8.3)
- vagrant (Tested against version 2.2.10)
- virtualbox
- docker
- a working vpn connection
